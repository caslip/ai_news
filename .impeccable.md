# AI News Aggregator Design Context

## Design Context

### Users
- **Primary**: 内容创作者、AI 爱好者
- **Context**: 需要快速获取 AI 领域资讯、撰写相关文章
- **Jobs**: 资讯浏览、文章创作、草稿管理
- **Emotions**: 追求效率、专业感、简洁

### Brand Personality
- **Voice**: 简洁、专业、高效
- **Tone**: 中性偏技术，清晰直接
- **3-word**: 专业、高效、简约

### Aesthetic Direction
- **Visual**: shadcn/ui new-york 风格，现代简洁
- **Theme**: 浅色模式为主，支持深色模式
- **Colors**: 中性色调 + 蓝色强调 (#3b7dd8)
- **Typography**: Inter/System fonts, 清晰可读

### Design Principles
1. **简洁优先** - 最小化 UI 复杂度，直达目标
2. **一致性** - 遵循 shadcn/ui 设计规范
3. **安全性** - API 密钥加密存储
4. **可扩展性** - 支持多种 AI 提供商
